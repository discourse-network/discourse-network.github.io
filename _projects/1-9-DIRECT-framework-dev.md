The [DIRECT framework](https://directframework.com/) is a framework of skills and competencies for dRTPs. A **central component** of making this framework available to the community is the web app, which so far has been developed by a collection of volunteers using their own time. The web app provides an accessible way to read through the framework and understand the skills, competencies, and categories it contains. It also provides users with a way to generate and visualise their own skill profiles in a skills wheel.

Continued development of this web app is an essential component of **DisCouRSE Work Package 3: Advanced Skills**. This is because the Web App can be used by dRTP leaders to develop themselves and their teams, and because the advanced training will be aligned to certain competencies in **DIRECT**.

Since this web app has been developed by volunteers based at different institutions, the progress and project management on it has been inconsistent. This project will secure dedicated professional time from the [Imperial RSE Team](https://www.imperial.ac.uk/admin-services/ict/self-service/research-support/rcs/service-offering/research-software-engineering/about-the-team/) to develop the web app, adding to the existing volunteer effort. This will result in a more robust infrastructure set-up, a thorough database, a better long-term roadmap, and improved sustainability of the web app.
